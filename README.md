# Full Stack To-Do Application

This is a Full Stack To-Do Application built using MERN stack.
In this users can sign up, log in and manage their tasks.
Each user can access their own todos.

## Tech Stack 
- Frontend : React, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB

## Features
- Singup and Login
- CRUD Todos
- User-specific task management
